# AI Resume Analyzer

A Python-based AI Resume Analyzer utilizing Streamlit for the interface and NLP (Natural Language Processing) for parsing and analyzing resumes.

## Features
- **Upload Resume**: Support for PDF files.
- **Extract Info**: Extracts contact details, skills, and education (using basic NLP).
- **Keyword Matching**: Compare resume against potential keywords (simple AI).
- **Match Score**: Calculate a match percentage against a job description.

## Tech Stack
- **Python 3.10+**
- **Streamlit** (Web UI)
- **Spacy** (NLP)
- **PyPDF2** or **pdfminer.six** (Text Extraction)
